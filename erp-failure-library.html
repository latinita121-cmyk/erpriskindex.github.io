ERP Failure Library

This library documents publicly reported ERP implementation failures using primary evidence sources only, including court filings, government audit reports, and Tier-1 financial media. Entries record outcomes and reported facts without asserting intent, fault, or misconduct beyond cited sources.

Evidence Standards

  • Primary sources only (court records, government audits, official filings)
  • Tier-1 financial media where primary documents are unavailable
  • No vendor opinions, marketing material, or blog sources
  • Neutral language focused on outcomes, costs, and timelines

Scope Notes

This library is not exhaustive. Entries are added only when verifiable documentation meets the evidence standards defined above. Absence of an organization or system does not imply success or failure.

This database records documented outcomes only. It does not assert intent, fault, or misconduct beyond cited sources.
