Updates

ERP Risk Index is updated only when documentation meets the evidence standards described in the Methodology page.

Get notified when new entries are added

For now, updates are distributed manually. Email: contact@erpriskindex.com

Subject line suggestion: “Updates request”

Submit corrections or sources

If you have an official document (audit report, filing, court record, regulator notice) that meets the evidence standards, email it to: contact@erpriskindex.com

Subject line suggestion: “Correction / Source submission”

Corrections Policy

  • If a source is corrected or updated, the database will update the entry and note the change date.
  • If a claim cannot be supported by eligible evidence, it is removed.

This database records documented outcomes only. It does not assert intent, fault, or misconduct beyond cited sources.
